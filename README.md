# Ansible Role: NFS


Installs NFS client on Debian/Ubuntu.

## Requirements

None.

## Role Variables

Available variables are listed below, along with default values (see `defaults/main.yml`):

    path_to_src:
 
Where will be mounted the nfs directory

    src_mnt:
    
Source of the nfs


## Dependencies

None.
